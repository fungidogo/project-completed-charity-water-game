# Water charity game project
A game where you have to click a certain amount of cans before the timer ends

It has 3 difficulties

Easy: Collect 20 cans in 50 seconds
Medium: Collect 30 cans in 40 seconds
Hard: Collect 40 cans in 30 seconds

The game uses HTML, CSS, and Javascript to function

A challenge I solved was figuring out how to add event listeners to each one of the grid cells and have them remove the water can from the cell when clicked.

I learned how to create and remove DOM elements using javascript, as well as functions such as setInterval() and querySelector()
I also learned how to create objects and arrays in javascript.
